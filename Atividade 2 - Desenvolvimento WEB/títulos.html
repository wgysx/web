HTML: A Linguagem da Web

A internet é composta por diversas tecnologias, e uma das principais é o HTML.

O que é HTML?

HTML significa HyperText Markup Language, que define a estrutura básica de uma página da web.

Principais Elementos do HTML

Títulos e Parágrafos

Os títulos vão de <h1> a <h6>. Parágrafos usam <p>.

Listas

Podem ser ordenadas (<ol>) ou não ordenadas (<ul>).

Links e Imagens

Links usam <a> e imagens usam <img>.

Importância da Hierarquia

A hierarquia dos títulos melhora a organização e acessibilidade do conteúdo.

Conclusão

Aprender HTML é essencial para criar páginas bem estruturadas.
