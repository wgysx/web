A Linguagem da Web

A internet é composta por diversas tecnologias, e uma das principais é o HTML. Ele permite a estruturação das páginas web, organizando conteúdos de forma clara e acessível.

O que é HTML?

HTML significa HyperText Markup Language. Ele define a estrutura básica de uma página da web, utilizando elementos chamados de "tags".

Principais Elementos

Títulos: Usam as tags <h1> até <h6>.

Parágrafos: Usam a tag <p>.

Listas: Ordenadas (<ol>) ou não ordenadas (<ul>).

Links: Usam a tag <a>.

Imagens: Usam a tag <img>.

Texto Formatado

Podemos usar sublinhado e negrito para destaque.

Exemplo de fórmula: E = mc².

Exemplo de fórmula química: H₂O.

Exemplo de tag HTML: <p>.
